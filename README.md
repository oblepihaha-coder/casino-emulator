# casino-emulator
Вебзастосунок, що емулює роботу онлайн-казино з можливістю реєстрації користувачів, керування балансом, здійснення ставок у реальному часі та перегляду історії ігор.

## Технологічний стек

* **Back-End:** C# (.NET Core), ASP.NET Core Web API, Entity Framework Core, SQLite (БД), JWT Auth.
* **Front-End:** Vue.js, Pinia (керування станом), Vue Router (маршрутизація), Bootstrap / Bootswatch (стилізація), Axios (HTTP-клієнт), AOS (анімації).

## Структура проєкту

* `/Backend` — серверна частина (контролери, моделі даних `User` та `Bet`, міграції SQLite).
* `/Frontend` — клієнтська частина застосунку на Vue.js.
  * `/src/components` — перевикористовувані елементи інтерфейсу.
  * `/src/views` — головні сторінки (вхід, реєстрація, ігровий зал).
  * `/src/stores` — збереження стану авторизації (Pinia) та робота з токенами.
  * `/src/interceptors` — перехоплювачі Axios для автоматичного додавання JWT-токенів.
  * `index.html` — головна точка входу Front-End.
